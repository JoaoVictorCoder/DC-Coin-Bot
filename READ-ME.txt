Cloudflared Tunnel Setup Guide for Coin Project (TXT Compatible)

This guide explains how to install Cloudflared, generate cert.pem and the tunnel credentials file (example: coin-bot.json), locate these files, and place them into the project's cloudflared folder. All steps include simple explanations.

------------------------------------------------------------
1. Install Cloudflared
------------------------------------------------------------

Windows (PowerShell):
    winget install --id Cloudflare.cloudflared

Linux (Debian/Ubuntu):
    curl -L https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64.deb -o cloudflared.deb
    sudo dpkg -i cloudflared.deb

macOS (Homebrew):
    brew install cloudflared

------------------------------------------------------------
2. Authenticate Cloudflared and Generate cert.pem
------------------------------------------------------------

Run this command:
    cloudflared login

A browser window opens asking you to select your domain.
After authorization, the cert.pem file is automatically created.

Common locations:
Windows:
    C:\Users\<User>\.cloudflared\cert.pem

Linux / macOS:
    /home/<user>/.cloudflared/cert.pem

Copy cert.pem into your project folder:
    /project-root/cloudflared/cert.pem

------------------------------------------------------------
3. Create a New Tunnel
------------------------------------------------------------

Use this command to create a tunnel with a chosen name:
    cloudflared tunnel create coin-bot

Cloudflared will display something like:

Tunnel created with ID: 12345678-abcd-1234-abcd-1234567890ab
Credentials file saved to: /home/user/.cloudflared/12345678-abcd.json

Copy this JSON file into:
    /project-root/cloudflared/coin-bot.json

------------------------------------------------------------
4. Locate the Generated Files
------------------------------------------------------------

You need two files:
1. cert.pem
2. The tunnel credentials JSON file, such as coin-bot.json

Cloudflared shows the file paths after each command. Copy both files into:

project-root/
    cloudflared/
        cert.pem
        coin-bot.json

------------------------------------------------------------
5. Create config.yml (Recommended)
------------------------------------------------------------

Inside the project's cloudflared folder, create config.yml:

tunnel: 12345678-abcd-1234-abcd-1234567890ab
credentials-file: cloudflared/coin-bot.json

ingress:
  - hostname: api.yourdomain.com
    service: http://localhost:3000
  - service: http_status:404

Replace the tunnel ID with the one Cloudflare generated.

------------------------------------------------------------
6. Run the Tunnel
------------------------------------------------------------

Option 1:
    cloudflared tunnel --config=cloudflared/config.yml run

Option 2:
    cloudflared tunnel run coin-bot

------------------------------------------------------------
7. Verify the Tunnel
------------------------------------------------------------

After running the tunnel, requests to your hostname will route to your local service.

Example:
    https://api.yourdomain.com/

------------------------------------------------------------
Summary of Commands
------------------------------------------------------------

Install:
    sudo apt install cloudflared

Login (generates cert.pem):
    cloudflared login

Create tunnel:
    cloudflared tunnel create coin-bot

Run tunnel:
    cloudflared tunnel run coin-bot

Run with config:
    cloudflared tunnel --config=cloudflared/config.yml run

------------------------------------------------------------
Final Notes
------------------------------------------------------------

- cert.pem authenticates your Cloudflare account.
- The tunnel JSON file authenticates a specific tunnel.
- Place both files in the project's cloudflared folder.
- Keep these files private.

Your Coin Project is now ready to use Cloudflare secure tunneling.
